Tutoriel : Installation du pack Shiiou 2025

Avant l'installation

  1. Téléchargez le pack Shiiou 2025

  2. Chemins d'installation :
    • Steam : <Lecteur où est installé Steam>\SteamLibrary\steamapps\common\Grand Theft Auto V
    • Rockstar Games Launcher : <Lecteur où est installé Rockstar>\Games\Grand Theft Auto V
    • Epic Games : <Lecteur où est installé Epic>\Epic Games\GTAV

  3. Logiciels nécessaires :
    • Un utilitaire de décompression comme WinRAR ou 7-Zip.

Assurez-vous que votre PC est assez puissant pour qu'il puisse le faire tourner

Étapes d'installation pour FiveM

1. Accéder au dossier FiveM

Pressez Win + R pour ouvrir une boîte de dialogue.

Tapez %localappdata%\FiveM\FiveM.app et validez.

2. Placer les fichiers

  • Dossier mods : Copiez les fichiers .rpf du pack.
  • Dossier plugins : Copiez l’intégralité des fichiers.

3. Vérifications

Assurez-vous que chaque fichier est bien au bon emplacement.

Étapes d'installation pour GTA V

1. Accéder au dossier d'installation

Repérez le chemin du jeu en fonction de votre launcher :

  • Steam :
    <Lecteur où est installé Steam>\SteamLibrary\steamapps\common\Grand Theft Auto V
  • Rockstar Games Launcher :
    <Lecteur où est installé Rockstar>\Games\Grand Theft Auto V
  • Epic Games :
    <Lecteur où est installé Epic>\Epic Games\GTAV

2. Placer les fichiers

Copiez l'intégralité des fichiers.

Conseils

  1. Sauvegardez vos fichiers originaux avant de copier les fichiers du pack.

  2. Si le jeu ne démarre pas :
    • Vérifiez les fichiers copiés et leur emplacement.

  3. Optimisez les performances en ajustant les paramètres graphiques via le jeu.
